## 📂 Features
AnyTransfer provides several useful features for users:

- ⚡ **Instant Access Codes**: Obtain access codes for your files immediately after selection.
- 📤 **Direct Storage Uploads**: Files are uploaded directly through signed URLs without server delays.
- 🔐 **GitHub Login**: Anonymous users can upload files up to 100MB, while logged-in users can transfer up to 1GB.
- 🛡️ **File Review System**: An admin panel allows for oversight, and users can report files as needed.
- 🎨 **User-Friendly Interface**: Enjoy a visually appealing design with smooth animations and a dark mode.
- 🌐 **Global Deployment**: Access the service from anywhere, thanks to the Cloudflare Edge Network.

## 🛠️ Technical Stack
AnyTransfer is built using reliable technologies, including:

- **Runtime**: Cloudflare Workers, providing edge computing capabilities.
- **Storage**: Cloudflare R2, a S3-compatible object storage system.
- **Metadata**: Cloudflare KV for efficient key-value storage.
- **Users**: Cloudflare D1, utilizing an SQLite database for user management.
- **Auth**: GitHub OAuth combined with JWT for secure logins.
